# Ooo-custom-map
A short custom map made for the game Öoo

How to play the map/mod:

1 Go to the location of the game files under SteamLibrary\steamapps\common\Ooo.

2 Replace the win.data file with this one.

3 Clear your save data using the in game menu and re open the game.

4 Your'e done.

(When replacing the data.win file saving the original one is suggested)
